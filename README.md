## Quick Start

```bash
npm install -g @vuepress-something/cli
```

```bash
vps init
```

如果你想在现有的项目里使用，可以这么做：
```bash
npm install vuepress vuepress-theme-something --save-dev
```

通过 `config.js` 使用主题：
```js
module.exports = {
  theme: 'vuepress-theme-something'
}
```
